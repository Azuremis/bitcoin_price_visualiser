# Bitcoin price visualiser

This notebook is a visualisation of bitcoin price data using the [quandl api](https://www.quandl.com/tools/api) and [plotly library](https://github.com/plotly/plotly.py).

Weighted price data from the exchanges below is used for aggregation and calculation of the average price.

The average price chart can be viewed interactively [here](https://chart-studio.plotly.com/~Azuremis/3/#plot). The period covered is from 13th September 2011 to 13th March 2021.

Exchanges:
- Coinbase
- Okcoin
- Bitstamp
- Kraken
- Bitfinex
- Itbit

To see the specific datasets, refer to [bitcoin charts exchange rate data](https://www.quandl.com/data/BCHARTS-Bitcoin-Charts-Exchange-Rate-Data).
